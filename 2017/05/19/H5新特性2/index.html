H5新特性2

  • H5新特性

H5中表单的新特性：
1，新的input type
2，新的表单元素
3，表单元素的新属性


一，新的input type
h4中的 input type：text（文本）、password（密码）、radio（单选）、checkbox（复选）、file（文件上传）、hidden（隐藏域）、submit（提交）、reset（）、image（）
h5中的新 input type：email（邮件）、url（连接的字段）、number（数字）、tel（电话）、search（搜索）、range（滑动条）、color（拾色器）、date（日期）、month（月份）、week（周）

二、新的表单元素
H4中的表单元素：input、option/select（下拉边框）、textarea（多行文本）、lable（与别的元素绑定，然后跳转）
H5中的新表单元素：datalist、progress、meter、output
1，datalist
<datalist>
<option>XX</option>
<option>YY</option>
</datalist>
datalist本身不可见，datalist为input的建议列表，用户可以选取，也可以自己输入。

2，progress
progress表示进度条：
<progress></progress> 表示一个 左右晃动的进度条
<progress value=".7"></progress> 具有指定进度的进度条

3，meter
Meter：度量衡，刻度尺，用于标示一个值的所处范围：优秀（绿色），可接受（黄色）不可接受（绿色）
<meter min="可取的最小值" max="可取的最大值" low="合理的下限" high="合理的上限值" optimum="最小值" value=“当前实际值”></meter>

4，ouput
输出，没有任何样式，样式上等同于span

三、H5表单的新属性
1，placeholder：占位符。
<input value="tom" placeholder="请输入用户名">
2，autofocus：自动获取焦点。
<input autofocus>
3，multiple：允许输入框输入多个输入（用逗号分隔）
<input type="email" multiple>
4，form：用于把输入域放到form外。
<form id="f5"></form>
<input form="f5">
———–输入验证相关新属性——————
5，required：必填项，内容不能为空
<input required>
6，maxlength：指定字符串的最大长度
<input maxlength="9">
7，minlength：指定字符串的最小长度
<input minlength="6">
8，max：指定数字最大值
<input max="60">
9，min：指定数字最小值
<input min="18">
10，pattern：指定输入必须符合的正则表达式
<input pattern="1[35789]\d{9}">
上述验证属性会影响表单元素对应的JS对象的validity属性。
validity 布尔属性值，返回 input 输入值是否合法

结语：H5的增强型表单给力很多新东西，简化了我们的js编写，但是，作为web来说兼容性问题才是最大的boss，而H5现在在兼容性方面还有所欠缺，还有，样式方面H5也无法更改，所有受不了H5丑陋的同学还是自己写吧。
