node+express+ejs项目搭建

写在前面

因为新开了一个hexo博客，所以不由的想要看看，hexo是怎么实现的，然后接着就发现默认的render插件（渲染插件）所用的模板引擎是ejs，所以我们就来看看怎么在后台使用ejs渲染页面。
然后，这里我们使用的后台是node+express，不知道的同学可以去看看http://www.expressjs.com.cn/ ，以后有空我们也再讲。

在浏览器中打开 http://localhost:3000/ 网址就可以看到这个应用了。

文件目录

接着我们来看看express-generator生成的项目文件目录：

bin: 启动目录 里面包含了一个启动文件 www 默认监听端口是 3000
node_modules: 所有的依赖模块
public: 所有的前端静态资源 html css image js
routes: 路由文件
views: 主要放置 ejs 后端模板文件
app.js: 入口文件(主文件) 总路由
package.json: 配置文件

以上就是简单的ejs项目搭建。
